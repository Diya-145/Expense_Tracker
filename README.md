# 💰 Expense Tracker

A professional command-line Expense Tracker application built using Python. It helps users record daily expenses, organize them into categories, search records, generate statistics, and export data to CSV.

---

## ✨ Features

- ➕ Add Expense
- 📋 View Expenses
- 🔍 Search Expense
- 🗑️ Delete Expense
- 📊 Expense Statistics
- 📂 Category-wise Expenses
- 📅 Expense Date
- 📤 Export Expenses to CSV
- 💾 JSON Storage
- 🎨 Colorful CLI using Colorama

---

## 🛠️ Technologies Used

- Python
- JSON
- CSV
- File Handling
- Colorama

## ▶️ Installation

### Clone Repository

```bash

```

### Install Dependency

```bash
pip install colorama
```

### Run Project

```bash
python main.py
```
